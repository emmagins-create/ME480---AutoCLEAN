ME480 is a senior-level class at the University of Washington Seattle campus.
In the class, students were tasked with creating a project that amounts to a capstone project done in a much smaller timeframe. 
Students had to identify a problem, come up with a solution, and create the solution all in 10 weeeks. 
The solution had to be a mechatronics focused, in order to introduce ME students to electronics aspects they were previously unfamiliar with.
